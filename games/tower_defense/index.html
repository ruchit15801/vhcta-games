💎 GAME MECHANICS

  • Earn Gold by defeating enemies
  • Upgrade towers by tapping/clicking them
  • Each tower has 3 upgrade levels
  • You have 20 Lives — don't let enemies reach the base!
  • Complete waves faster for Time Bonus Gold
  • Special abilities recharge every 3 waves
  • 10 Levels of increasing difficulty

🎮 CONTROLS

  • Select Tower → Click tower panel → Click empty grid cell
  • Upgrade/Sell → Click placed tower
  • Speed Up → ⚡ button (2x speed)
  • Pause → ⏸ button
  • Special Ability → 💥 button (screen nuke)
